# EasyColoc

## À propos
Easy Coloc résout un problème très évident et simple : la traçabilité.

Easy Coloc offre une plateforme que les colocataires peuvent utiliser comme grand livre comptable central.
Toutes les transactions impliquant les résidents y sont enregistrées, puis réparties équitablement entre eux.
La plateforme fournit un historique complet de tous vos paiements et leur traçabilité, ainsi qu'un système de réputation qui vous permet de voir les actions passées de vos colocataires.

## Fonctionnalités

### Utilisateur
- Gestion de la colocation
- Gestion des dépenses
- Gestion des catégories
- Invitation via le système de messagerie électronique

### Administrateur
- Bannissement d'utilisateurs
- Promotion d'utilisateurs
- Statistiques de base
